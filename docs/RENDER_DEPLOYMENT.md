# Render Deployment

Set backend Render env vars:
- DATABASE_URL
- FRONTEND_URL
- OPENAI_API_KEY
- OPENAI_MODEL_BLUEPRINT
- STRIPE_SECRET_KEY
- STRIPE_WEBHOOK_SECRET
- STRIPE_PRICE_PRO_MONTHLY
- STRIPE_PRICE_ELITE_MONTHLY
- STRIPE_PRICE_FOUNDERS_LIFETIME
- JWT_SECRET_KEY
- ENVIRONMENT=production
- ALLOW_TEST_TIER_ACCESS=false

Set frontend Render env vars:
- VITE_API_BASE_URL
- VITE_STRIPE_PUBLISHABLE_KEY
- VITE_ALLOW_TEST_TIER_ACCESS=false
